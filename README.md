# 00:13

トランスフォーマーG1／サウンドウェーブを相手にした、ブラウザ用の短編テキストゲームです。  
主人公は無個性・無言の夢主で、プレイヤーは選択肢を選ぶことで進行します。

黒い端末に表示される文字だけを頼りに、0時13分から始まる通信と反復をたどってください。  
1周は短めです。いくつかの結末があります。

## 内容
- サウンドウェーブ夢
- 主人公は無個性・無言
- 選択肢による進行
- 文字のみ／短編

## プレイ方法
ページを開き、選択肢をクリックまたはタップして進めてください。  
セーブ機能はブラウザ依存です。環境によっては保持されない場合があります。
